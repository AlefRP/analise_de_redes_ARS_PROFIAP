# 🌐 Análise de Redes Sociais no Mestrado em Administração Pública (PROFIAP) com Pajek

## 📝 Descrição do Projeto
Este projeto foi desenvolvido como parte do Mestrado em Administração Pública (PROFIAP) na Universidade Federal de Alfenas (UNIFAL-MG), focando na análise de redes sociais utilizando o software Pajek. O objetivo principal é explorar as interações e conexões entre os alunos do programa de pós-graduação e outras amostras relevantes.

## 🎯 Objetivos
- Analisar as relações e redes sociais dentro do grupo acadêmico.
- Identificar padrões de interação e colaboração entre os alunos.

## 🛠️ Ferramentas Utilizadas
- **[Pajek](http://mrvar.fdv.uni-lj.si/pajek/)**: Software para análise de redes sociais.

## 📊 Amostra de Dados
- Alunos do programa de pós-graduação em Administração Pública.
- Outras amostras relevantes para a análise.
